version:
1.1.0
2017/11/1
===================================================

# Install guide:

For installing fhi-aims plugin for xcrysden, at first "install xcrysden" , then:

	tar zxvf aims2xsf_v1.1.0.tar.gz
	cd aims2xsf
        chmod +x ./install
        ./install

===================================================
# Run 											      
 $ aims2xsf geometry.in
